Version 8.15 (2004-09-22)

This is the first release of GPL Ghostscript in the new version 8.1x series, and the fifth release in that series over all. This release offers a very large number of bug fixes over AFPL Ghostscript 8.14, and many significant new features over the previous GPL Ghostscript 8.01. We recommend upgrading to all our free users.

New features in this version :

  • The PDF writer now implements PDF 1.4 encryption. See the encryption parameter documentation for more information.
  • Ghostscript now better supports running as a job server, were multiple files can be piped to a single instance. Just invoke Ghostscript with the -dJOBSERVER and pipe jobs to stdin, delimiting them with ^D.
  • Font rendering has be improved over previous releases.

Incompatible changes

There are no known incompatible changes from the previous release.
